My first css project
We all love cats. They are also a fun way to learn all the HTTP status codes. Your task is to refactor the given html and to create a template to represent each cat card block on it’s own. After you have created the templete render it into the div with id “allCats”. A cat has an id, statusCode, statusMessage and imageLocation. The cats are seeded using the function from the js file named catSeeder.js Each card block has a button that unveils status code information connected to each cat. You should toggle the button and change it’s text from “Show status code” to “Hide status code”.
Днес денят ми започна добре- събудих се в 7ч и се чуствах добре. Имах много интересен сън- апокалиптичен. Бях с Мария на някакво игрище; бяхме и с още някаква двойка, имаше и други хора. Не съм сигурен дали действието се развиваше във Франция или в България. Изведнъж някакви много наситено черни облаци се появиха на небето недалече от нас. Казах, че веднага трябва да тръгваме. Не ме взеха на сериозно. Изведнъж се забеляза някаква вълна на разрушение. По пътя си тя поразяваше всичко- имаше експлозии, хора умираха... Настана паника. Стана ясно, че няма как да избягаме- вълната се приближаваше прекалено бързо. Застанахме измежду някакви сгради недалеч от самото игрище- нямаше смисъл да бягаме повече, а да се крием в сградите беше още по безмислено, защото вълната като че ли поразяваше най-вече тях и те биваха изпепелявани. Започнахме да се молим (или поне аз). Не ни оставаше много. Обаче изведнъж всичко спря като по чудо. След това се озовах „вкъщи“- май беше във Франция, но не в апартамента, а в къща, която за първи път виждах, но знаех, че е наша. Имаше напукани прозорци, отвън се виждаше съвсем не много щети. Погледнах през единия прозорец и се виждаше морето. Беше по скоро мрачно, буреносно. Светкавица разцепи небето чак до хоризонта.... Интересен сън наистина, един от най-интереснито, които съм имал. Бърз преглед в интернет показа, че ще има голяма промяна в живота ми. Бях готов за всичко, което Бог ми е отредил. Или поне така мисля...
